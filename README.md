### Objetivo

Letra musica HASD em quadros visiveis

Cifra Melodica

**Iniciar a fazer as bases das cifras melódicas**

- Colocar controle na Estrofe de aparecer/ocultar a cifra melododica

- Controle só deve estar habilitado caso tenha cifra melodica

- Fazer busca por nomes e parte dos nomes também

- Retornar a lista contendo os nomes ou parte dos nomes, ao clicar no item da lista vai para os versos
